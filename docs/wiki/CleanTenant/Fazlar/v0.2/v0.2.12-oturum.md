# Oturum — v0.2.12 (2026-05-21)

## Hedef

Kapsamlı kullanıcı yönetimi revizyonu: ilk giriş şifre zorlaması, OTP şifre sıfırlama, onboarding wizard, scope-bazlı aktif/pasif kontrolü ve tenant'ın tüm alt kapsamlarındaki kullanıcıları tek ekranda görmesi.

## Tamamlanan 9 Görev

- [x] **A — NavMenu yeniden yapılanması** — System context'te Tenant/Company linklerini `Yönetimler` grubuna taşıdı; Tenant context'te `Kullanıcılar` linki `_canManageUsers` ile gate'lendi
- [x] **B — RequiresPasswordChange** — `User` entity'e bool property eklendi; EF migration (`20260521133910`); login handler'da `IssuePasswordChangeChallengeAsync`; `CompletePasswordChangeCommand`; `ChangePasswordPage.razor`
- [x] **C — Redis OTP servisi** — `IVerificationCodeService` + `RedisVerificationCodeService` (key: `{prefix}:otp:{key}`, 6 hane, single-use)
- [x] **D — Şifre sıfırlama akışı** — `RequestPasswordResetCommand` (enumeration korumalı) + `ResetPasswordWithCodeCommand`; `__ct_pwd_rst` cookie; `/forgot-password` + `/reset-password` sayfaları; Login'e "Şifremi unuttum" linki
- [x] **E — LookupUserByIdentifierQuery** — TCKN/VKN/Phone/Email ile gerçek zamanlı kullanıcı arama; debounced 500ms
- [x] **F — UserOnboardingWizard.razor** — `Idle → Searching → UserFound / NewUser` state machine; UserFound: mevcut kullanıcıya rol ata; NewUser: yeni kullanıcı oluştur; UserListPanel Tenant toolbar'ında tek "Kullanıcı Ekle" butonu
- [x] **G — Scope-bazlı deactivate/reactivate** — `DeactivateTenantUserCommand` / `ReactivateTenantUserCommand`; `UserRoleAssignment.IsActive` toggle (global `User.IsActive` değil); `UserListPanel` scope-aware
- [x] **H — Tenant kapsamlı kullanıcı görünümü** — `ListUsersQueryHandler`: `isTenantView = Scope==Tenant && TenantId.HasValue` → Tenant-scope + Company-scope atamaları birleştir; `UserListItem.IsActive` tenant içi aktif atamadan türet
- [x] **I — Build + doğrulama** — 0 hata / 0 uyarı; `/forgot-password`, `/reset-password`, `/login` HTTP 200; localization seeder 120 yeni anahtar ekledi

## Önemli Kararlar

- **`User.IsActive` vs `UserRoleAssignment.IsActive`:** Global ban ile scope-bazlı deaktivasyon ayrıldı. Tenant admin yalnız kendi kapsamındaki atamayı pasifleştirir; kullanıcı diğer tenant'larında etkilenmez.
- **Enumeration koruması:** `RequestPasswordResetCommand` kullanıcı bulunamasa bile `Result.Success` döner; `/forgot-password` her zaman `/reset-password`'a yönlendirir.
- **Tenant kapsamlı görünüm:** `ScopeLevel.Tenant` + `TenantId` kombinasyonunda hem tenant-scope hem company-scope atamalar sorgulanır; `UserListItem.IsActive` global flag yerine aktif atama varlığından türetilir.
- **Password change challenge store:** İlk giriş şifre zorlaması için `IPasswordChangeChallengeStore` (Redis, 15 dk TTL) — 2FA challenge pattern'iyle örtüşür; finalize'da `LoginFinalizer.FinalizeAsync` çağrılır.
